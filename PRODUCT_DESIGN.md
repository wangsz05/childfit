## 📖 第一部分：产品概述

### 1.1 产品定位

| 维度 | 定义 |
|------|------|
| **产品名称** | ChildFit - 看天安排孩子运动的公益 App |
| **Slogan** | "雾霾天不知道让孩子干嘛？用它！" |
| **副 Slogan** | "所有功能永久免费，农村城市同等可用" |
| **产品形态** | 微信小程序 (主) + H5(辅) |
| **目标用户** | 3-18 岁儿童青少年家长，尤其关注农村/留守儿童 |
| **核心价值** | 天气动态适配 + 运动护眼组合 + 公益普惠 |

### 1.2 产品愿景

> **让每个孩子——无论城市农村、富裕贫困——都能获得科学的健康指导，养成受益终身的运动习惯。**

### 1.3 公益承诺

| 承诺 | 说明 |
|------|------|
| 🆓 **永久免费** | 所有功能永久免费，无付费墙、无广告 |
| 🌍 **普惠可及** | 农村/城市、富裕/低收入家庭同等可用 |
| 📚 **专业可靠** | 内容经专家审核，不因为是公益而降低质量 |
| 🔍 **透明运营** | 资金使用、合作伙伴、影响力数据公开 |
| 🤝 **开放协作** | 代码/内容开源，欢迎各方参与 |
| 🌱 **长期主义** | 不追求短期爆发，注重长期影响力 |

### 1.4 核心差异化

| 差异化 | 说明 | 竞品状态 |
|--------|------|----------|
| 🌤️ **天气动态适配** | 雾霾/雨天自动室内，晴天优先户外 | 所有竞品❌ |
| 👁️ **运动 + 护眼组合** | 每日户外 2 小时 + 眼保健操 + 视力追踪 | 整合同步❌ |
| 📅 **作息智能联动** | 上学日/周末/考试周/疫情居家自动适配 | 所有竞品❌ |
| 🎗️ **公益 + 普惠** | 永久免费 + 农村适配 + 特殊儿童关怀 | 商业竞品❌ |

### 2.2 核心功能详细设计

#### 2.2.1 天气适配引擎 (P0+)

**功能描述**: 根据实时天气和预报，动态调整活动推荐

**天气规则矩阵**:

| 天气类型 | 预警级别 | 推荐策略 | 安全提示 |
|----------|----------|----------|----------|
| 晴 | 无 | 户外活动优先 | 注意防晒 |
| 晴 | 高温 35-40°C | 清晨/傍晚户外，限时 30 分钟 | 防暑降温 |
| 晴 | 高温 40°C+ | 取消户外运动 | 高温预警 |
| 多云 | 无 | 户外活动优先 | - |
| 阴 | 无 | 户外活动正常 | - |
| 小雨 | 无 | 室内活动优先，或雨具户外 | 防滑 |
| 中雨/大雨 | 无 | 室内活动 | - |
| 暴雨 | 蓝/黄预警 | 室内活动 | 避免外出 |
| 暴雨 | 橙/红预警 | 取消运动，安全第一 | 灾害预警 |
| 雾霾 | AQI 50-150 | 减少户外，敏感人群室内 | 戴口罩 |
| 雾霾 | AQI 150+ | 室内活动 | 污染预警 |
| 雪 | 小雪 | 室内活动优先 | 防滑保暖 |
| 雪 | 中雪/大雪 | 室内活动 | 避免外出 |
| 雷电 | 预警中 | 室内活动 | 雷电危险 |
| 台风 | 任意级别 | 取消户外运动 | 台风预警 |
| 沙尘暴 | 任意级别 | 室内活动 | 关闭窗户 |

**天气突变处理**:

| 场景 | 触发条件 | 处理方式 |
|------|----------|----------|
| 出门前突变 | 计划前 1 小时天气变化 | 推送提醒："建议改为 XX 室内活动" |
| 运动中突变 | 实时监测到雷电/暴雨 | 紧急通知："请立即停止户外运动" |
| 预报不准 | 用户反馈天气不符 | 记录反馈，优先使用实时数据 |

**技术实现**:
- 数据源：和风天气 (主) + 心知天气 (备)
- 更新频率：每 30 分钟自动刷新
- 缓存策略：同地区同条件复用，TTL 1 小时
- 降级方案：API 故障时使用本地规则引擎

#### 2.2.4 打卡系统 (P0)

**打卡类型**:

| 类型 | 方式 | 说明 |
|------|------|------|
| 手动打卡 | 点击完成 | 最简单，适合 MVP |
| 拍照打卡 | 上传照片 | AI 识别 (后期) |
| 视频打卡 | 上传短视频 | AI 识别动作 (后期) |
| 语音打卡 | 语音描述 | 老人友好 |

**打卡流程**:

```
用户选择活动 → 执行活动 → 点击打卡
                              ↓
                        选择打卡方式
                              ↓
                    手动/拍照/视频/语音
                              ↓
                        记录完成
                              ↓
                        获得奖励
```

**激励设计**:

| 激励类型 | 示例 | 目的 |
|----------|------|------|
| 即时反馈 | "太棒了！完成今日运动！🎉" | 正向强化 |
| 连续打卡 | 7 天/30 天徽章 | 习惯养成 |
| 成就系统 | "运动小达人"等级 | 长期激励 |
| 成长养成 | 虚拟宠物/成长树 | 游戏化 |
| 家长表扬 | "今天孩子很棒！"推送 | 亲子互动 |

#### 2.2.6 饮食推荐系统 (P1)

**营养需求计算**:

| 年龄段 | 每日热量需求 | 蛋白质 | 钙 |
|--------|--------------|--------|-----|
| 3-6 岁 | 1200-1400 kcal | 30-40g | 600mg |
| 7-9 岁 | 1600-1800 kcal | 45-55g | 800mg |
| 10-12 岁 | 2000-2200 kcal | 60-70g | 1000mg |
| 13-18 岁 | 2400-2800 kcal | 75-85g | 1200mg |

**饮食推荐原则**:

| 原则 | 说明 |
|------|------|
| 🌾 **顺时而食** | 春生、夏长、秋收、冬藏，应季食材优先 |
| 💰 **平价可及** | 本地常见食材，避免昂贵进口食品 |
| 🌍 **地域适配** | 南方/北方、城市/农村、沿海/内陆差异 |
| 🍽️ **营养均衡** | 参考《中国居民膳食指南 (2022) 儿童版》 |
| ⚠️ **安全警示** | 食物过敏原提示、禁忌说明 |
| 🎗️ **公益免费** | 所有饮食建议永久免费 |

**气候 - 饮食规则矩阵**:

| 气候条件 | 饮食原则 | 推荐食材 | 禁忌食材 |
|----------|----------|----------|----------|
| 高温 (>35°C) | 清热、补水、开胃 | 绿豆、黄瓜、西瓜、苦瓜 | 油腻、辛辣、冷饮 |
| 低温 (<5°C) | 保暖、高能量 | 羊肉、牛肉、姜、坚果 | 生冷、寒凉 |
| 高湿 (>80%) | 祛湿、健脾 | 薏米、赤小豆、冬瓜、陈皮 | 甜腻、生冷 |
| 干燥 (<30%) | 润燥、滋阴 | 梨、银耳、百合、蜂蜜 | 辛辣、油炸 |
| 雾霾 (AQI>150) | 清肺、排毒、抗氧化 | 梨、木耳、海带、西兰花 | 烧烤、油炸 |
| 雷雨/台风 | 安神、易消化 | 小米、莲子、百合、牛奶 | 生冷、油腻 |

**季节饮食推荐**:

| 季节 | 饮食原则 | 推荐食材 | 食谱示例 |
|------|----------|----------|----------|
| 春 | 养肝、健脾、防过敏 | 菠菜、韭菜、山药、蜂蜜 | 菠菜猪肝汤、山药粥 |
| 夏 | 清热、祛湿、补水 | 绿豆、冬瓜、薏米、西瓜 | 绿豆汤、冬瓜排骨汤 |

| 冬 | 保暖、补肾、高能量 | 羊肉、牛肉、黑豆、核桃 | 羊肉萝卜汤、黑豆粥 |
