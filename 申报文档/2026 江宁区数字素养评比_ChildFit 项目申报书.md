## 🎯 作品简介

### 一、项目背景

随着城市化进程加快和电子产品普及，我国儿童青少年面临两大健康挑战：

1. **运动不足**：据《中国儿童青少年身体活动指南》，6-17 岁儿童青少年每日应进行至少 60 分钟中高强度身体活动，但实际达标率不足 30%
2. **视力下降**：2022 年全国儿童青少年总体近视率达到 53.6%，户外活动时间不足是重要诱因
3. **天气影响**：雾霾、雨天等恶劣天气时，家长不知道如何安排孩子的室内活动

同时，农村和留守儿童群体缺乏专业的运动健康指导资源，城乡健康服务差距明显。

### 二、作品概述

**ChildFit** 是一款基于 Web 的智能儿童运动健康推荐系统，核心功能包括：

- 🌤️ **天气动态适配**：根据实时天气（AQI、降雨、温度等）自动推荐室内/户外运动
- 👁️ **运动 + 护眼组合**：每日户外 2 小时 + 眼保健操，预防近视
- 📅 **作息智能联动**：自动识别上学日/周末/考试周，生成适配计划
- 🎗️ **公益普惠**：所有功能永久免费，农村城市同等可用

### 三、核心创新点

| 创新维度 | 具体说明 |
|----------|----------|
| **技术创新** | 基于多源数据（天气 API+ 学生作息 + 年龄特征）的智能推荐算法 |
| **模式创新** | 公益免费模式，无付费墙、无商业广告 |
| **服务创新** | 城乡同等服务，关注留守儿童健康 |
| **功能创新** | 运动 + 护眼三维联动（天气 + 作息 + 护眼） |

### 三、核心算法

#### 1. 天气适配推荐算法

```python
def recommend_activity(user_profile, weather_data, schedule):
    """
    根据天气、用户档案、作息生成推荐活动
    
    参数:
        user_profile: 用户档案（年龄、健康状况、偏好）
        weather_data: 天气数据（AQI、温度、降雨、风力）
        schedule: 作息信息（上学日/周末/考试周）
    
    返回:
        推荐活动列表（含时间、强度、注意事项）
    """
    # AQI 判断
    if weather_data.aqi > 150:
        activity_type = "室内"
    elif weather_data.rain_probability > 0.5:
        activity_type = "室内"
    else:
        activity_type = "户外"
    
    # 年龄段适配
    if user_profile.age <= 6:
        intensity = "低强度"
        duration = 30
    elif user_profile.age <= 12:
        intensity = "中强度"
        duration = 60
    else:
        intensity = "中高强度"
        duration = 90
    
    # 作息适配
    if schedule.type == "考试周":
        duration *= 0.5
        activity_type = "轻度活动"
    
    return generate_activity_list(activity_type, intensity, duration)
```

```python
def calculate_eye_protection_score(outdoor_hours, eye_exercise_count):
    """

    规则:
        - 每日户外 2 小时 = 100 分
        - 每次眼保健操 = 20 分
        - 连续 7 天达标 = 额外 50 分
    """
    base_score = min(outdoor_hours / 2.0, 1.0) * 100
```
