# 🍲 气候适配饮食推荐方案

**文档版本**: v1.0  
**创建时间**: 2026-03-26  
**基于**: 28 轮迭代成果 + 气候饮食专项设计

---

## 📖 概述

### 设计理念

> **"顺时而食，因地制宜"** - 根据气候、季节、地域、天气，提供科学、平价、可及的饮食建议。

### 核心原则

| 原则 | 说明 |
|------|------|
| 🌾 **顺时而食** | 春生、夏长、秋收、冬藏，应季食材优先 |
| 💰 **平价可及** | 本地常见食材，避免昂贵进口食品 |
| 🌍 **地域适配** | 南方/北方、城市/农村、沿海/内陆差异 |
| 🍽️ **营养均衡** | 参考《中国居民膳食指南 (2022) 儿童版》 |
| ⚠️ **安全警示** | 食物过敏原提示、禁忌说明 |
| 🎗️ **公益免费** | 所有饮食建议永久免费 |

## 🌡️ 第一部分：气候维度饮食推荐

### 1.1 温度维度

#### 高温天气 (>35°C)

| 需求 | 推荐食材 | 食谱示例 | 注意事项 |
|------|----------|----------|----------|
| **补水** | 西瓜、黄瓜、冬瓜、绿豆 | 绿豆汤、西瓜汁、凉拌黄瓜 | 避免冰镇过度 |
| **清热** | 苦瓜、莲子、百合、薄荷 | 苦瓜炒蛋、莲子百合粥 | 脾胃虚寒少吃苦瓜 |
| **开胃** | 番茄、柠檬、山楂、酸奶 | 番茄蛋汤、柠檬水 | 避免过酸伤胃 |
| **蛋白质** | 鸭肉、鱼肉、豆腐、鸡蛋 | 清蒸鱼、冬瓜鸭肉汤 | 避免油腻 |
| **电解质** | 香蕉、椰子水、淡盐水 | 香蕉奶昔、椰子水 | 运动后补充 |

**高温食谱示例**:
```
早餐：绿豆粥 + 凉拌黄瓜 + 鸡蛋 + 馒头
午餐：清蒸鱼 + 炒苦瓜 + 冬瓜汤 + 米饭
晚餐：番茄鸡蛋面 + 凉拌豆腐
加餐：西瓜 (适量)、酸奶、柠檬水
```

**注意事项**:
- ⚠️ 避免大量冷饮 (伤脾胃)
- ⚠️ 避免油腻食物 (难消化)
- ⚠️ 运动后补充淡盐水 (防脱水)
- ✅ 少量多餐 (减轻肠胃负担)

---

#### 低温天气 (<5°C)

| 需求 | 推荐食材 | 食谱示例 | 注意事项 |
|------|----------|----------|----------|
| **保暖** | 羊肉、牛肉、姜、葱、蒜 | 羊肉萝卜汤、姜汤 | 适量即可，避免上火 |
| **高能量** | 坚果、芝麻、花生、糯米 | 芝麻糊、花生粥 | 控制量，避免肥胖 |
| **维生素** | 胡萝卜、南瓜、红薯、菠菜 | 胡萝卜炒肉、南瓜粥 | 深色蔬菜优先 |
| **蛋白质** | 鸡肉、猪肉、鱼、豆制品 | 红烧鸡肉、豆腐炖鱼 | 温热烹饪方式 |
| **汤水** | 鸡汤、排骨汤、菌菇汤 | 山药排骨汤、菌菇鸡汤 | 温热饮用 |

**低温食谱示例**:
```
早餐：小米粥 + 煮鸡蛋 + 馒头 + 芝麻酱
午餐：羊肉萝卜汤 + 炒菠菜 + 米饭
晚餐：鸡肉炖蘑菇 + 红薯粥 + 馒头
加餐：核桃、花生、热牛奶
```

**注意事项**:
- ⚠️ 避免生冷食物 (伤阳气)
- ⚠️ 火锅适量 (避免上火)
- ✅ 温热烹饪 (蒸、煮、炖)
- ✅ 热汤热水 (保暖)

### 1.2 湿度维度

#### 高湿天气 (梅雨/回南天)

| 需求 | 推荐食材 | 食谱示例 | 注意事项 |
|------|----------|----------|----------|
| **祛湿** | 薏米、赤小豆、冬瓜、陈皮 | 薏米赤小豆粥、陈皮冬瓜汤 | 孕妇慎用薏米 |
| **健脾** | 山药、莲子、芡实、白术 | 山药莲子粥、芡实汤 | 健脾助消化 |
| **清淡** | 青菜、豆腐、鱼肉 | 清蒸鱼、炒青菜 | 避免油腻 |
| **开胃** | 陈皮、山楂、柠檬 | 陈皮水、山楂糕 | 适量即可 |

**高湿食谱示例**:
```
早餐：薏米赤小豆粥 + 煮鸡蛋 + 馒头
午餐：清蒸鱼 + 炒冬瓜 + 山药排骨汤 + 米饭
晚餐：豆腐青菜汤 + 莲子炒肉 + 米饭
加餐：陈皮水、山楂片
```

**注意事项**:
- ⚠️ 避免生冷、甜腻食物 (加重湿气)
- ⚠️ 少吃水果 (寒凉生湿)
- ✅ 温热饮食 (助阳祛湿)

---

#### 干燥天气 (秋燥/北方冬季)

| 需求 | 推荐食材 | 食谱示例 | 注意事项 |
|------|----------|----------|----------|
| **润燥** | 梨、银耳、百合、蜂蜜 | 冰糖炖梨、银耳百合羹 | 蜂蜜 1 岁以下禁用 |
| **滋阴** | 芝麻、核桃、豆浆、牛奶 | 芝麻糊、核桃露 | 适量即可 |
| **润肺** | 莲藕、白萝卜、枇杷、杏仁 | 莲藕排骨汤、杏仁露 | 杏仁适量 |
| **补水** | 黄瓜、番茄、柚子、橙子 | 鲜榨果汁、蔬菜沙拉 | 常温饮用 |

**干燥食谱示例**:
```
早餐：豆浆 + 芝麻糊 + 煮鸡蛋 + 馒头
午餐：莲藕排骨汤 + 炒百合 + 米饭
晚餐：冰糖炖梨 + 清蒸鱼 + 青菜 + 米饭
加餐：银耳羹、柚子、蜂蜜水
```

**注意事项**:
- ⚠️ 避免辛辣、油炸 (加重干燥)
- ⚠️ 少吃烧烤 (燥热伤阴)
- ✅ 多喝水 (少量多次)
- ✅ 炖、煮、蒸为主 (保留水分)

### 1.3 空气质量维度

#### 雾霾天 (AQI>150)

| 需求 | 推荐食材 | 食谱示例 | 科学依据 |
|------|----------|----------|----------|
| **清肺** | 梨、百合、银耳、罗汉果 | 冰糖炖梨、百合银耳羹 | 润肺止咳 |
| **排毒** | 黑木耳、海带、绿豆、芹菜 | 凉拌木耳、绿豆汤 | 吸附有害物质 |
| **抗氧化** | 蓝莓、番茄、胡萝卜、西兰花 | 番茄炒蛋、西兰花炒肉 | 对抗自由基 |
| **增强免疫** | 香菇、灵芝、黄芪、枸杞 | 香菇鸡汤、枸杞粥 | 提升抵抗力 |
| **Omega-3** | 深海鱼、亚麻籽、核桃 | 清蒸鱼、核桃露 | 抗炎作用 |

**雾霾天食谱示例**:
```
早餐：绿豆粥 + 凉拌木耳 + 煮鸡蛋 + 馒头
午餐：清蒸鱼 + 炒西兰花 + 海带豆腐汤 + 米饭
晚餐：冰糖炖梨 + 香菇鸡汤 + 青菜 + 米饭
加餐：蓝莓、核桃、罗汉果茶
```

**注意事项**:
- ⚠️ 减少户外活动时间
- ⚠️ 关闭窗户，使用空气净化器
- ✅ 多喝水 (促进代谢)
- ✅ 多吃深色蔬菜水果 (抗氧化)

### 1.4 特殊天气维度

#### 雷雨/台风天

| 需求 | 推荐食材 | 食谱示例 | 注意事项 |
|------|----------|----------|----------|
| **安神** | 小米、莲子、百合、牛奶 | 小米粥、莲子百合羹 | 缓解紧张情绪 |

| **维生素 C** | 橙子、柠檬、猕猴桃、青椒 | 鲜榨橙汁、青椒炒肉 | 增强免疫力 |

```
早餐：小米粥 + 蒸蛋 + 馒头

晚餐：青菜面条 + 豆腐 + 橙子
加餐：牛奶、饼干 (储备食品)
```

**注意事项**:
- ⚠️ 提前储备食材 (应对可能的供应中断)
- ⚠️ 避免生冷食物 (雷电天气肠胃敏感)
- ✅ 温热易消化饮食
- ✅ 保持心情平静

## 🌸 第二部分：季节维度饮食推荐

### 2.1 春季 (3-5 月)

**气候特点**: 温暖、多风、花粉多、湿气渐增

**饮食原则**: 春生、养肝、健脾、防过敏

| 需求 | 推荐食材 | 食谱示例 |
|------|----------|----------|
| **养肝** | 菠菜、韭菜、芹菜、枸杞 | 菠菜猪肝汤、韭菜炒蛋 |
| **健脾** | 山药、莲子、薏米、大枣 | 山药粥、莲子红枣汤 |
| **防过敏** | 蜂蜜、胡萝卜、西兰花、鱼 | 蜂蜜水、清蒸鱼 |
| **应季蔬菜** | 春笋、荠菜、香椿、豌豆苗 | 春笋炒肉、香椿炒蛋 |

**春季一周食谱示例**:
```
周一：菠菜猪肝汤 + 韭菜炒蛋 + 米饭
周二：春笋炒肉 + 荠菜豆腐汤 + 米饭
周三：清蒸鱼 + 炒西兰花 + 山药粥
周四：香椿炒蛋 + 芹菜炒肉 + 米饭
周五：枸杞鸡肉粥 + 炒豌豆苗 + 馒头
周六：红枣莲子汤 + 炒时蔬 + 米饭
周日：蜂蜜水 + 胡萝卜炒肉 + 米饭
```

**注意事项**:
- ⚠️ 过敏体质避免花粉多的食物
- ⚠️ 少吃酸味 (收敛不利肝气疏泄)
- ✅ 多吃甘味食物 (健脾)
- ✅ 多吃绿色蔬菜 (养肝)

---

### 2.2 夏季 (6-8 月)

**气候特点**: 炎热、多雨、湿度大、日照长

**饮食原则**: 夏长、清热、祛湿、补水

| 需求 | 推荐食材 | 食谱示例 |
|------|----------|----------|
| **清热** | 苦瓜、冬瓜、黄瓜、绿豆 | 苦瓜炒蛋、绿豆汤 |
| **祛湿** | 薏米、赤小豆、冬瓜、陈皮 | 薏米赤小豆粥 |
| **补水** | 西瓜、番茄、黄瓜、椰子水 | 西瓜汁、番茄蛋汤 |
| **开胃** | 番茄、柠檬、山楂、酸奶 | 番茄蛋汤、柠檬水 |
| **应季水果** | 西瓜、桃子、李子、葡萄 | 西瓜、桃子 (适量) |

**夏季一周食谱示例**:
```
周一：绿豆粥 + 凉拌黄瓜 + 清蒸鱼 + 米饭
周二：苦瓜炒蛋 + 冬瓜排骨汤 + 米饭
周三：番茄蛋汤 + 炒青菜 + 薏米粥
周四：凉拌豆腐 + 西瓜 + 鸡肉粥
周五：冬瓜鸭肉汤 + 炒青菜 + 米饭
周六：柠檬水 + 清蒸鱼 + 黄瓜炒肉 + 米饭
周日：赤小豆粥 + 凉拌木耳 + 米饭
```

**注意事项**:
- ⚠️ 避免大量冷饮 (伤脾胃)
- ⚠️ 少吃烧烤、油炸 (上火)
- ✅ 清淡饮食为主
- ✅ 少量多餐 (减轻肠胃负担)

### 2.3 秋季 (9-11 月)

**气候特点**: 凉爽、干燥、温差大

**饮食原则**: 秋收、润燥、滋阴、养肺

| 需求 | 推荐食材 | 食谱示例 |
|------|----------|----------|

| **养肺** | 莲藕、白萝卜、枇杷、杏仁 | 莲藕排骨汤、杏仁露 |
| **应季水果** | 梨、苹果、葡萄、柚子 | 冰糖炖梨、鲜榨果汁 |
| **应季蔬菜** | 莲藕、南瓜、红薯、菠菜 | 南瓜粥、红薯粥 |

**秋季一周食谱示例**:
```
周一：冰糖炖梨 + 莲藕排骨汤 + 米饭
周二：银耳百合羹 + 清蒸鱼 + 米饭
周三：芝麻糊 + 炒菠菜 + 南瓜粥
周四：蜂蜜水 + 白萝卜炖肉 + 米饭
周五：核桃露 + 炒青菜 + 红薯粥
周六：杏仁露 + 清蒸鱼 + 苹果
周日：豆浆 + 炒时蔬 + 米饭
```

**注意事项**:
- ⚠️ 避免辛辣、油炸 (加重秋燥)
- ⚠️ 少吃烧烤 (燥热伤阴)
- ✅ 多喝水 (少量多次)
- ✅ 多吃白色食物 (养肺)

---

```
周六：核桃露 + 白菜猪肉饺子
周日：花生粥 + 菠菜猪肝汤 + 馒头
```

**注意事项**:

### 3.1 南方地区

```
早餐：薏米赤小豆粥 + 肠粉 + 豆浆
```

### 3.2 北方地区

#### 北方干燥 (北京、河北、山西等)

| 特点 | 饮食策略 | 推荐食材 |
|------|----------|----------|
| 干燥寒冷 | 润燥保暖 | 梨、银耳、百合、羊肉 |
| 口味偏咸 | 控制盐分 | 减少酱油、咸菜 |
| 面食为主 | 粗细搭配 | 馒头、面条 + 杂粮 |
| 蔬菜较少 | 补充维生素 | 大白菜、萝卜、土豆 (冬季) |

**北方干燥食谱示例**:
```
早餐：小米粥 + 煮鸡蛋 + 馒头 + 芝麻酱
午餐：羊肉萝卜汤 + 炒白菜 + 米饭/馒头
晚餐：冰糖炖梨 + 猪肉炖粉条 + 馒头
加餐：银耳羹、苹果、热牛奶
```

---

#### 北方冬季严寒 (东北、内蒙古等)

| 特点 | 饮食策略 | 推荐食材 |
|------|----------|----------|
| 严寒 | 高能量保暖 | 羊肉、牛肉、坚果 |
| 腌制菜 | 补充维生素 | 酸菜、泡菜 (注意亚硝酸盐) |
| 炖菜文化 | 温热驱寒 | 猪肉炖粉条、小鸡炖蘑菇 |
| 室内暖气 | 注意补水 | 多喝水、多吃水果 |

**北方严寒食谱示例**:
```
早餐：杂粮粥 + 煮鸡蛋 + 馒头 + 咸菜 (少量)
午餐：羊肉炖萝卜 + 酸菜猪肉 + 米饭
晚餐：小鸡炖蘑菇 + 猪肉炖粉条 + 馒头
加餐：核桃、花生、热牛奶、水果
```

### 3.3 西部地区

#### 高原地区 (西藏、青海、四川西部)

| 特点 | 饮食策略 | 推荐食材 |
|------|----------|----------|
| 高寒缺氧 | 高能量、易消化 | 青稞、牦牛肉、酥油 |
| 紫外线强 | 抗氧化 | 番茄、胡萝卜、蓝莓 |
| 干燥 | 润燥补水 | 梨、蜂蜜、银耳 |
| 藏族饮食 | 尊重习惯 | 糌粑、酥油茶、青稞酒 (儿童不宜) |

**高原食谱示例**:
```
早餐：青稞粥 + 酥油茶 (少量) + 煮鸡蛋
午餐：牦牛肉炖萝卜 + 炒青菜 + 米饭
晚餐：番茄鸡蛋汤 + 青稞饼 + 炒时蔬
加餐：梨、蜂蜜水、核桃
```

#### 西北干旱 (新疆、甘肃、宁夏等)

| 特点 | 饮食策略 | 推荐食材 |
|------|----------|----------|
| 干旱少雨 | 润燥补水 | 哈密瓜、葡萄、梨 |
| 牛羊肉多 | 适量食用 | 羊肉串 (少量)、手抓肉 |
| 面食为主 | 粗细搭配 | 馕、拉面 + 杂粮 |
| 瓜果丰富 | 多吃水果 | 葡萄、哈密瓜、西瓜 |

**西北食谱示例**:
```
早餐：奶茶 (少量) + 馕 + 煮鸡蛋
午餐：手抓羊肉 (适量) + 炒青菜 + 米饭
晚餐：番茄鸡蛋面 + 哈密瓜
加餐：葡萄、梨、酸奶
```

### 3.4 农村地区专属

**农村饮食特点**: 自给自足、时令食材、成本低

| 优势 | 利用策略 |
|------|----------|
| 自家菜园 | 多吃新鲜蔬菜 |
| 散养家禽 | 鸡蛋、鸡肉 (适量) |
| 应季水果 | 自家果树水果 |
| 杂粮丰富 | 玉米、红薯、土豆 |

**农村平价食谱示例**:
```
早餐：红薯粥 + 煮鸡蛋 + 馒头
午餐：土豆烧肉 + 炒青菜 + 玉米饭
晚餐：豆腐青菜汤 + 红薯 + 馒头
加餐：自家水果、花生、瓜子
```

**注意事项**:
- ✅ 充分利用本地食材 (降低成本)
- ✅ 多吃新鲜蔬菜 (自家菜园)
- ⚠️ 注意食物储存 (避免霉变)
- ⚠️ 荤素搭配 (避免营养不良)

## 👶 第四部分：年龄段饮食推荐

### 4.1 幼儿组 (3-6 岁)

**营养需求**: 热量 1200-1400kcal/天，蛋白质 30-40g/天

| 重点 | 推荐食材 | 注意事项 |
|------|----------|----------|
| **易消化** | 粥、面条、蒸蛋、豆腐 | 食物切碎煮烂 |
| **补钙** | 牛奶、酸奶、豆腐、小鱼干 | 每天 300-500ml 奶 |
| **补铁** | 猪肝、瘦肉、菠菜、黑木耳 | 搭配维生素 C 促进吸收 |
| **补锌** | 牡蛎、瘦肉、核桃、花生 | 坚果碾碎防呛 |
| **维生素** | 各种蔬菜水果 | 多样化，颜色丰富 |

**幼儿一日食谱示例**:
```
早餐：牛奶 (200ml) + 蒸蛋 + 小馒头
加餐：苹果泥/香蕉
午餐：肉末豆腐 + 炒青菜 + 软米饭
加餐：酸奶 (100g)
晚餐：鱼肉粥 + 炒胡萝卜 + 小馒头
```

---

### 4.2 小学低年级 (7-9 岁)

**营养需求**: 热量 1600-1800kcal/天，蛋白质 45-55g/天

| 重点 | 推荐食材 | 注意事项 |
|------|----------|----------|
| **生长发育** | 牛奶、鸡蛋、鱼肉、瘦肉 | 优质蛋白质 |
| **补脑** | 核桃、芝麻、鱼、蛋黄 | DHA、卵磷脂 |
| **护眼** | 胡萝卜、菠菜、蓝莓、玉米 | 维生素 A、叶黄素 |
| **骨骼** | 牛奶、豆腐、小鱼干 | 钙 + 维生素 D |

**小学低年级一日食谱示例**:
```
早餐：牛奶 (250ml) + 煮鸡蛋 + 全麦面包 + 水果
加餐：核桃 2-3 个
午餐：清蒸鱼 + 炒菠菜 + 米饭 + 紫菜汤
加餐：酸奶 (150g)
晚餐：瘦肉炒胡萝卜 + 豆腐汤 + 米饭
```

---

### 4.3 小学高年级 (10-12 岁)

**营养需求**: 热量 2000-2200kcal/天，蛋白质 60-70g/天

| 重点 | 推荐食材 | 注意事项 |
|------|----------|----------|
| **青春期前** | 高蛋白、高钙 | 为发育储备 |
| **补脑** | 深海鱼、核桃、芝麻 | 学习压力大 |
| **护眼** | 胡萝卜、蓝莓、菠菜 | 预防近视 |
| **运动后** | 蛋白质 + 碳水 | 运动后 30 分钟内补充 |

**小学高年级一日食谱示例**:
```
早餐：牛奶 (300ml) + 煮鸡蛋 2 个 + 全麦面包 + 水果
加餐：核桃 3-4 个
午餐：红烧牛肉 + 炒西兰花 + 米饭 + 番茄蛋汤
加餐：酸奶 (200g) + 香蕉
晚餐：清蒸鱼 + 炒青菜 + 米饭 + 豆腐汤
```

---

### 4.4 青少年 (13-18 岁)

**营养需求**: 热量 2400-2800kcal/天，蛋白质 75-85g/天

| 重点 | 推荐食材 | 注意事项 |
|------|----------|----------|
| **生长发育高峰** | 高蛋白、高钙、高铁 | 男孩女孩需求不同 |
| **女孩经期** | 补铁、补血 | 猪肝、瘦肉、菠菜 |
| **运动量大** | 高能量、高蛋白 | 运动后及时补充 |
| **学习压力** | 补脑、减压 | 深海鱼、坚果、牛奶 |

**青少年一日食谱示例 (男)**:
```
早餐：牛奶 (400ml) + 煮鸡蛋 2 个 + 全麦面包 + 水果
加餐：坚果一小把
午餐：红烧牛肉/羊肉 + 炒青菜 + 米饭 + 汤
加餐：酸奶 (250g) + 香蕉
晚餐：清蒸鱼 + 炒肉片 + 米饭 + 豆腐汤
夜宵 (如学习晚): 热牛奶 + 饼干
```

**青少年一日食谱示例 (女)**:
```
早餐：牛奶 (300ml) + 煮鸡蛋 + 全麦面包 + 水果
加餐：核桃 3-4 个
午餐：瘦肉炒菠菜 + 米饭 + 汤
加餐：酸奶 (200g) + 水果
晚餐：清蒸鱼 + 炒青菜 + 米饭
经期额外：红糖姜茶、猪肝汤
```

## ⚠️ 第五部分：饮食禁忌与安全

### 5.1 常见过敏原

| 过敏原 | 常见食物 | 替代方案 |
|--------|----------|----------|
| **牛奶** | 牛奶、酸奶、奶酪 | 豆浆、杏仁奶、燕麦奶 |
| **鸡蛋** | 鸡蛋、蛋糕、面条 | 豆腐、肉类蛋白质 |
| **花生** | 花生、花生酱 | 其他坚果 (如不过敏) |
| **坚果** | 核桃、杏仁、腰果 | 种子类 (葵花籽、南瓜籽) |
| **海鲜** | 鱼、虾、蟹、贝类 | 肉类、豆制品 |
| **大豆** | 豆腐、豆浆、酱油 | 其他豆类、肉类 |
| **小麦** | 面包、面条、馒头 | 米饭、玉米、土豆 |

---

### 5.2 食物相克 (科学看待)

| 传言 | 科学解释 | 建议 |
|------|----------|------|
| 虾 + 维生素 C=砒霜 | 剂量远达不到中毒 | 可同食 |
| 菠菜 + 豆腐=结石 | 草酸钙沉淀，但量少 | 菠菜焯水后食用 |
| 鸡蛋 + 豆浆=不消化 | 无科学依据 | 可同食 |
| 牛奶 + 橘子=腹泻 | 个体差异，非普遍 | 观察个体反应 |

**科学建议**: 多样化饮食，适量即可，无需过度担心。

---

### 5.3 特殊人群饮食禁忌

| 人群 | 禁忌 | 建议 |
|------|------|------|
| **哮喘儿童** | 冷饮、辛辣、海鲜 (如过敏) | 温热饮食，避免诱发 |
| **肥胖儿童** | 油炸、甜食、含糖饮料 | 低热量、高纤维 |
| **糖尿病儿童** | 高糖食物、精制碳水 | 低 GI 食物，定时定量 |
| **肾病儿童** | 高盐、高蛋白 | 低盐、适量优质蛋白 |
| **心脏病儿童** | 高盐、高脂 | 低盐低脂，少量多餐 |

## 📊 第六部分：饮食推荐算法设计

### 6.1 推荐逻辑

```
用户档案 (年龄/地区/健康状况/经济状况)
        ↓
气候数据 (季节/温度/湿度/空气质量)
        ↓
饮食规则引擎 (匹配气候 + 季节 + 地域)
        ↓
过滤禁忌 (过敏原/疾病禁忌)
        ↓
成本分级 (免费/低/中/高)
        ↓
生成食谱 (早餐/午餐/晚餐/加餐)
        ↓
用户反馈 (👍/👎) → 优化推荐
```

### 6.2 气候 - 饮食规则矩阵

| 气候条件 | 饮食原则 | 推荐食材 | 禁忌食材 |
|----------|----------|----------|----------|
| 高温 (>35°C) | 清热、补水、开胃 | 绿豆、黄瓜、西瓜、苦瓜 | 油腻、辛辣、冷饮 |
| 低温 (<5°C) | 保暖、高能量 | 羊肉、牛肉、姜、坚果 | 生冷、寒凉 |
| 高湿 (>80%) | 祛湿、健脾 | 薏米、赤小豆、冬瓜、陈皮 | 甜腻、生冷 |
| 干燥 (<30%) | 润燥、滋阴 | 梨、银耳、百合、蜂蜜 | 辛辣、油炸 |
| 雾霾 (AQI>150) | 清肺、排毒、抗氧化 | 梨、木耳、海带、西兰花 | 烧烤、油炸 |
| 雷雨/台风 | 安神、易消化 | 小米、莲子、百合、牛奶 | 生冷、油腻 |
