## 🔄 迭代轮次 23 - 经济差异场景

### ❓ 关键问题
- 低收入家庭买不起运动器材怎么办？
- 付费兴趣班 (游泳/舞蹈) 推荐是否公平？
- 付费营养食品 (进口奶粉/有机食品) 是否造成焦虑？
- 如何避免"推荐的都是有钱家庭才能玩的"?

### 💡 洞察 (发现公平性问题!)

**原设计可能无意中造成"经济歧视"!**

| 活动类型 | 成本 | 原设计是否考虑 | 公平性风险 |
|----------|------|----------------|------------|
| 跑步/跳绳 | 低 (<50 元) | ✅ | 低 |
| 自行车 | 中 (300-1000 元) | ❌ | 中 |
| 游泳 | 高 (课程 + 装备>2000 元/年) | ❌ | 高 |
| 舞蹈/钢琴 | 高 (课程>5000 元/年) | ❌ | 高 |
| 滑雪/冰球 | 极高 (>10000 元/年) | ❌ | 极高 |
| 进口营养品 | 高 | ❌ | 高 |

**经济分层活动库设计**:

| 成本等级 | 定义 | 活动示例 | 占比目标 |
|----------|------|----------|----------|
| **免费** | 0 元 | 跑步、徒手健身、公园游玩 | 50%+ |
| **低成本** | <100 元 | 跳绳、毽子、橡皮筋 | 30%+ |
| **中成本** | 100-500 元 | 自行车、羽毛球拍 | 15% |
| **高成本** | >500 元 | 游泳课、舞蹈课 | 5% (可选) |

**饮食推荐公平性**:

| 原推荐 | 问题 | 改进方案 |
|--------|------|----------|
| 三文鱼 (富含 Omega-3) | 贵，农村买不到 | 替代：淡水鱼/鸡蛋 |
| 进口奶粉 | 贵 | 替代：本地牛奶/豆浆 |
| 有机蔬菜 | 贵 | 替代：普通蔬菜 (营养相近) |
| 坚果零食 | 较贵 | 替代：花生/瓜子 (本地便宜) |

**经济状况选项**:

| 选项 | 推荐策略 |
|------|----------|
| 经济紧张 | 仅推荐免费/低成本活动，饮食用平价替代 |
| 经济一般 | 正常推荐，标注成本等级 |
| 经济宽裕 | 正常推荐，包含高成本选项 |

### 🔧 改进点 (发现公平性问题!)

- [ ] 活动成本标注 (免费/低/中/高)
- [ ] 经济状况选项 (影响推荐)
- [ ] 免费活动占比 50%+ (保证公平)
- [ ] 饮食平价替代方案
- [ ] 避免制造焦虑 (不推荐必须付费的内容)

## 🔄 迭代轮次 24 - 动机与心理场景

### ❓ 关键问题
- 孩子就是不愿意运动怎么办？
- 运动 3 天新鲜感过了如何坚持？
- 被同学嘲笑"太胖/太瘦/太弱"怎么办？
- 青春期孩子叛逆不听家长的话怎么办？
- 运动成瘾 (过度运动) 如何识别？

### 💡 洞察 (发现心理层面遗漏!)

**原设计假设"孩子愿意运动"，但现实是...**

| 心理问题 | 占比 (估计) | 原设计是否覆盖 | 需要支持 |
|----------|-------------|----------------|----------|
| 缺乏动机 | ~40% | ❌ | 动机激发策略 |
| 新鲜感消失 | ~60% | 部分 | 持续激励 |
| 身体形象焦虑 | ~20% | ❌ | 心理支持 |
| 青春期叛逆 | ~30% | ❌ | 沟通策略 |
| 运动成瘾 | ~5% | ❌ | 风险识别 |

**动机激发策略**:

| 动机类型 | 策略 |
|----------|------|
| 外在动机 (奖励) | 成就徽章、打卡奖励、小礼品 |
| 内在动机 (乐趣) | 游戏化设计、有趣的活动、自主选择 |
| 社交动机 (认同) | 朋友 PK、家长表扬、展示成果 |
| 意义动机 (目标) | 健康目标、成长记录、未来自我 |

**坚持困难应对**:

| 阶段 | 特征 | 应对策略 |
|------|------|----------|
| 第 1 周 | 新鲜感强 | 充分利用，建立习惯 |
| 第 2-3 周 | 新鲜感下降 | 增加变化，避免单调 |
| 第 1 个月 | 倦怠期 | 降低要求，允许休息 |
| 第 2-3 个月 | 习惯形成期 | 强化正向反馈 |
| 3 个月 + | 习惯稳定 | 适度挑战，避免自满 |

**身体形象焦虑支持**:

| 场景 | 支持方案 |
|------|----------|
| 被嘲笑太胖 | - 避免体重相关语言<br>- 强调健康而非外表<br>- 心理支持资源 |
| 被嘲笑太瘦 | - 避免外表评价<br>- 强调力量和健康<br>- 营养支持 |
| 被嘲笑太弱 | - 渐进式增强信心<br>- 记录进步而非比较<br>- 心理支持资源 |

**青春期叛逆应对**:

| 策略 | 说明 |
|------|------|
| 自主选择权 | 让孩子自己选活动，而非家长指定 |
| 隐私尊重 | 不公开分享孩子不愿分享的内容 |
| 平等沟通 | 解释为什么运动重要，而非强制 |
| 同伴影响 | 推荐朋友也在做的活动 |
| 专业背书 | 用专家/偶像的话，而非家长的话 |

**运动成瘾识别**:

| 警示信号 | 应对 |
|----------|------|
| 每天运动>3 小时 | 提醒休息，设置上限 |
| 受伤仍坚持运动 | 强制休息，就医建议 |
| 因运动影响学习/社交 | 提醒平衡，时间管理 |
| 不运动就焦虑 | 心理支持资源 |

### 🔧 改进点 (发现心理层面遗漏!)

- [ ] 动机激发系统 (多类型动机)
- [ ] 坚持困难应对 (分阶段策略)
- [ ] 身体形象焦虑支持 (避免伤害语言)
- [ ] 青春期模式 (自主选择 + 隐私)
- [ ] 运动成瘾识别 (警示信号 + 干预)
- [ ] 心理健康资源链接

## 🔄 迭代轮次 25 - 季节性疾病场景

### ❓ 关键问题
- 流感季如何调整推荐？
- 过敏季 (花粉/柳絮) 如何处理？
- 手足口病高发期如何建议？
- 新冠/传染病疫情期间如何应对？

### 💡 洞察 (发现健康风险遗漏!)

**原设计只考虑"健康状态"，但季节性疾病会影响!**

| 季节性疾病 | 高发期 | 影响 | 原设计是否覆盖 |
|------------|--------|------|----------------|
| 流感 | 冬春 (11 月 -3 月) | 避免人群聚集 | ❌ |
| 过敏 (花粉/柳絮) | 春季 (3-5 月) | 避免户外 | ❌ |
| 手足口病 | 春夏 (4-7 月) | 避免公共场地 | ❌ |
| 诺如病毒 | 秋冬 (10 月 -2 月) | 注意卫生 | ❌ |
| 新冠/传染病 | 不定期 | 根据防控政策 | 部分 |

**季节性疾病应对方案**:

| 疾病 | 高发期推荐调整 |
|------|----------------|
| **流感季** | - 避免人群聚集活动<br>- 室内活动优先<br>- 勤洗手提醒<br>- 疫苗接种提醒 |
| **过敏季** | - 花粉浓度高时避免户外<br>- 过敏体质专属提醒<br>- 室内空气净化建议<br>- 抗过敏药物准备 |
| **手足口病** | - 避免公共游乐场<br>- 家庭/私人场地优先<br>- 卫生消毒提醒 |
| **传染病疫情** | - 根据防控政策调整<br>- 居家运动方案<br>- 健康监测 |

**过敏体质专属选项**:

| 过敏原 | 规避策略 |
|--------|----------|
| 花粉 | 花粉浓度高时室内，关闭窗户 |
| 柳絮 | 柳絮季减少户外，戴口罩 |
| 尘螨 | 室内定期清洁，除螨 |
| 宠物皮屑 | 避免接触宠物，选择无宠物场地 |
| 霉菌 | 潮湿天气注意，保持干燥 |

### 🔧 改进点 (发现健康风险遗漏!)

- [ ] 季节性疾病提醒 (流感/过敏/手足口)
- [ ] 过敏体质选项 (专属规避策略)
- [ ] 公共卫生事件响应机制
- [ ] 疫苗接种提醒 (流感疫苗等)
- [ ] 与疾控信息联动 (疫情预警)

## 🔄 迭代轮次 26 - 发育阶段特殊需求

### ❓ 关键问题
- 青春期发育高峰期如何适配？
- 女孩月经期如何调整？
- 生长痛如何处理？
- 换牙期/恒牙期如何注意？
- 睡眠需求变化如何适配？

### 💡 洞察 (发现发育阶段遗漏!)

**原设计按年龄划分，但同龄孩子发育差异很大!**

| 发育阶段 | 特征 | 特殊需求 | 原设计是否覆盖 |
|----------|------|----------|----------------|
| 青春期前 | 生长稳定 | 正常推荐 | ✅ |
| 青春期启动 | 生长加速 | 营养增加，睡眠增加 | ❌ |
| 生长高峰期 | 年增高>10cm | 生长痛，易疲劳 | ❌ |
| 女孩月经期 | 周期性 | 经期调整运动 | ❌ |
| 变声期 | 嗓音变化 | 避免过度用嗓 | ❌ |

**青春期专属适配**:

| 变化 | 适配方案 |
|------|----------|
| 生长加速 | - 营养需求增加 (蛋白质/钙)<br>- 睡眠需求增加 (9-10 小时)<br>- 运动强度可适度增加 |
| 生长痛 | - 拉伸运动增加<br>- 按摩/热敷建议<br>- 避免过度负重 |
| 身体形象关注 | - 避免外表评价语言<br>- 强调健康而非外表<br>- 心理支持 |
| 情绪波动 | - 运动减压建议<br>- 情绪管理资源<br>- 家长沟通指导 |

**女孩经期专属**:

| 经期阶段 | 运动建议 |
|----------|----------|
| 经前 (1-3 天) | 正常运动，注意保暖 |
| 经中 (1-3 天) | 低强度运动，避免剧烈 |
| 经后 (4-7 天) | 逐渐恢复，补充营养 |
| 严重痛经 | 休息为主，热敷，就医建议 |

**睡眠需求适配**:

| 年龄段 | 推荐睡眠 | 原设计是否覆盖 |
|--------|----------|----------------|
| 3-5 岁 | 10-13 小时 | ❌ |
| 6-12 岁 | 9-12 小时 | ❌ |
| 13-18 岁 | 8-10 小时 | ❌ |

### 🔧 改进点 (发现发育阶段遗漏!)

- [ ] 青春期模式 (营养 + 睡眠 + 心理)
- [ ] 女孩经期专属调整
- [ ] 生长痛应对方案
- [ ] 睡眠需求提醒 (按年龄)
- [ ] 发育阶段自评问卷

## 🔄 迭代轮次 27 - 运动伤害与恢复场景

### ❓ 关键问题
- 运动受伤后如何恢复？
- 如何预防常见运动伤害？
- 运动后肌肉酸痛如何处理？
- 如何识别过度训练？

### 💡 洞察 (发现安全恢复遗漏!)

**原设计关注"运动前推荐"，但"运动后恢复"同样重要!**

| 伤害类型 | 常见原因 | 预防措施 | 原设计是否覆盖 |
|----------|----------|----------|----------------|
| 扭伤/拉伤 | 热身不足、动作错误 | 热身指导、动作规范 | ❌ |
| 擦伤/摔伤 | 场地不安全、装备不当 | 场地检查、装备建议 | ❌ |
| 肌肉酸痛 | 突然增加强度 | 渐进式增强、拉伸 | ❌ |
| 脱水/中暑 | 高温运动、补水不足 | 补水提醒、高温预警 | 部分 |
| 骨折 | 高冲击运动、意外 | 风险评估、安全提示 | ❌ |

**运动伤害预防体系**:

| 阶段 | 措施 |
|------|------|
| 运动前 | - 热身指导 (视频)<br>- 装备检查清单<br>- 场地安全检查 |
| 运动中 | - 正确动作指导<br>- 强度监控提醒<br>- 不适立即停止 |
| 运动后 | - 拉伸指导 (视频)<br>- 补水/营养补充<br>- 肌肉恢复建议 |

**常见伤害处理指南**:

| 伤害 | 紧急处理 | 就医建议 |
|------|----------|----------|
| 轻微扭伤 | RICE 原则 (休息/冰敷/加压/抬高) | 疼痛持续>3 天 |
| 擦伤 | 清洁消毒，创可贴 | 伤口深/污染严重 |
| 肌肉酸痛 | 休息，热敷，轻度拉伸 | 疼痛严重/影响活动 |
| 脱水 | 补充水分和电解质 | 意识模糊/无法饮水 |
| 中暑 | 阴凉处休息，降温 | 高烧/意识障碍 |
| 疑似骨折 | 固定，不要移动 | 立即就医 |

**恢复期运动推荐**:

| 恢复阶段 | 推荐活动 |
|----------|----------|
| 急性期 (1-3 天) | 休息为主，轻微活动 |
| 恢复期 (4-7 天) | 低强度活动，避免受伤部位 |
| 康复期 (1-2 周) | 渐进式恢复，功能训练 |
| 完全恢复 | 正常活动，注意预防 |

### 🔧 改进点 (发现安全恢复遗漏!)

- [ ] 热身指导视频 (每个活动前)
- [ ] 拉伸指导视频 (每个活动后)
- [ ] 常见伤害处理指南
- [ ] 恢复期运动推荐
- [ ] 过度训练识别 (警示信号)
- [ ] 急救知识普及

## 🔄 迭代轮次 28 - 长期可持续场景

### ❓ 关键问题
- 孩子长大后 (超 18 岁) 如何过渡？
- 项目停止服务后数据如何处理？
- 如何避免"用一年就不用了"?
- 如何让孩子养成终身运动习惯？

### 💡 洞察 (发现长期视角遗漏!)

**原设计关注"当下使用"，但"长期可持续"同样重要!**

| 长期问题 | 影响 | 原设计是否覆盖 | 应对方案 |
|----------|------|----------------|----------|
| 超龄用户 | 18 岁后无法使用 | ❌ | 成人版过渡/推荐替代 |
| 项目停止 | 服务终止 | ❌ | 数据导出/开源延续 |
| 用户流失 | 新鲜感消失 | 部分 | 持续内容更新/社交绑定 |
| 习惯未养成 | 停用后恢复旧习惯 | ❌ | 习惯养成体系 |

**超龄过渡方案**:

| 年龄段 | 方案 |
|--------|------|
| 16-17 岁 | 提醒即将超龄，推荐成人运动 App |
| 18 岁 + | 数据导出，推荐 Keep/咕咚等成人平台 |
| 终身运动习惯 | 总结运动偏好，提供个性化建议 |

**项目停止预案**:

| 措施 | 说明 |
|------|------|
| 提前通知 | 停止前 3 个月通知用户 |
| 数据导出 | 提供完整数据导出功能 |
| 开源延续 | 代码/内容开源，社区可接续 |
| 数据移交 | 移交给合作方 (学校/基金会) |
| 替代推荐 | 推荐类似公益项目 |

**习惯养成体系**:

| 阶段 | 目标 | 策略 |
|------|------|------|
| 第 1 个月 | 建立规律 | 固定时间，降低门槛 |
| 第 2-3 个月 | 形成习惯 | 正向反馈，适度挑战 |
| 第 4-6 个月 | 内化认同 | "我是爱运动的人" |
| 6 个月 + | 自主维持 | 减少外部激励，内在驱动 |

**终身运动习惯培养**:

| 要素 | 培养方式 |
|------|----------|
| 运动乐趣 | 多样化活动，避免单调 |
| 运动能力 | 渐进式提升，获得成就感 |
| 运动社交 | 朋友/家人一起参与 |
| 运动认同 | "运动是我生活的一部分" |

### 🔧 改进点 (发现长期视角遗漏!)

- [ ] 超龄过渡方案 (18 岁 + 推荐)
- [ ] 项目停止预案 (数据导出 + 开源)
- [ ] 习惯养成体系 (6 个月周期)
- [ ] 终身运动理念培养
- [ ] 用户退出调研 (了解流失原因)
