## 📋 如何参与

### 1. 代码贡献

**需要的技能**:
- 前端：微信小程序开发、Taro/uni-app、Vue/React
- 后端：Node.js、TypeScript、NestJS、PostgreSQL
- UI/UX：界面设计、图标设计、视觉设计
- 测试：测试用例编写、质量保证

**贡献流程**:
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request

### 2. 内容贡献

**需要的内容**:
- 儿童运动活动库 (按年龄段、天气、成本分类)
- 饮食食谱库 (按季节、气候、地域分类)
- 眼保健操视频/指导
- 健康科普文章
- 多语言翻译

**贡献方式**:
1. 在 Issues 中提出内容建议
2. 或直接提交 PR 添加内容文件
3. 内容需经专家审核后合并

### 3. 专家顾问

**需要的领域**:
- 儿科医生 (健康内容审核)
- 运动专家 (活动库审核)
- 营养师/注册营养师 (饮食建议审核)
- 眼科医生 (护眼内容审核)
- 心理专家 (心理健康内容)
- 中医师 (季节饮食指导)
