## 📖 项目简介

**ChildFit** 是一个公益性的儿童运动健康推荐系统，根据**天气**、**学生作息表**、**年龄段**，动态安排个性化的活动时间和饮食推荐。

**核心差异化** (所有竞品都没有的):
- 🌤️ **天气动态适配** - 雾霾/雨天自动室内，晴天优先户外
- 👁️ **运动 + 护眼组合** - 每日户外 2 小时 + 眼保健操
- 📅 **作息智能联动** - 上学日/周末/考试周自动适配
- 🎗️ **公益普惠** - 所有功能永久免费，农村城市同等可用

---

## 🎯 目标用户

- **主要**: 3-18 岁儿童青少年家长
- **重点关注**: 农村/留守儿童、低收入家庭
- **B 端**: 学校 (体育课补充工具)、公益组织

## 🏗️ 技术栈

**前端**: Vue 3 + Vite + Pinia
- 框架：Vue 3 (Composition API)
- 构建工具：Vite
- 状态管理：Pinia
- 路由：Vue Router 4
- HTTP 客户端：Axios

**后端**: 
- 语言：Python 3.12
- 框架：FastAPI
- 数据库：MySQL 8.0 + Redis
- ORM: SQLAlchemy 2.0
- 天气 API: 和风天气

**部署**: Docker + Docker Compose + 云服务器 (申请公益额度)
